# RunJS
RanJS是一个运行在Chrome上的插件，可以在打开一个网页时运行匹配到的代码块。比如自动矫正页面图片、删除小广告等。

# 安装使用
去Chrome Store下载安装 https://chrome.google.com/webstore/detail/runjs/ibhodfakinmhpplnphhbkmnhemaboama
